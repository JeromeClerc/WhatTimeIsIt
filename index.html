Whattimeisit

jQuery plugin for selecting time in a form

Download .zip Download .tar.gz View on GitHub

Demo

If your customers have to set an hour in a form. Use WhatTimeIsIt to set it the easiest way

Just click and select. That's easy.

Compatibility

With all the main browsers. Developped with jQuery 2.0.3.

The Code

Download the package and include it in your project this way. Do not forget the pics.

The HTML : 

<link rel="stylesheet" href="stylesheets/styles.css">

<form>
  <input type="text" name="time" class="wtii" placeholder="00:00">
</form>
The JS without parameters : 

<script>
  $(document).ready(function() {
    // By default, the script start at 12:00
    $('.wtii').whattimeisit();
  });
</script>
The JS witht parameters : 

<script>
  $(document).ready(function() {
    // The ampm variable allow the script to start hours at 0:00 or 12:00
    $('.wtii').whattimeisit({ampm : 'AM'});
  });
</script>
